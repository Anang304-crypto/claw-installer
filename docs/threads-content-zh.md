### 教學 4：常見錯誤排解

```
🔧 遇到問題？別慌，看這邊

整理了大家最常遇到的問題：

---

❌「Node.js 版本太舊」

→ 你的 Node.js 需要 22 以上
→ ClawInstaller 會告訴你怎麼更新
→ 通常就是裝一下 nvm，跑個指令

---

❌「Gateway not responding」

→ OpenClaw 的服務沒有啟動
→ 打開 ClawInstaller → Health Monitor
→ 按「Start」就好了

---

❌「API key 無效」

→ 可能複製的時候少了幾個字
→ 回去 Claude/OpenAI 的網站重新複製
→ 記得複製 完整 的 key

---

❌「Telegram bot 沒反應」

→ 確認 token 有貼對
→ 確認 Gateway 有在跑（看上面那點）
→ 試著重新啟動一次

---

還是搞不定？

→ 留言描述你的問題
```
